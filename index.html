La solución

Cuatro pilares.
Una red impecable.

Una arquitectura de red moderna y escalable, diseñada para el crecimiento de DOSP.

⚡

DHCP

Asignación automática de IPs en 4 subredes independientes. Sin configuración manual.

🛡

Active Directory

Dominio D2P.LAN para gestión centralizada de usuarios, grupos y permisos.

🌐

DNS

BIND9 + Windows Server con zonas directa, inversa y forwarders para Internet.

🔐

GPO

Directivas de Grupo personalizadas por departamento para máxima seguridad.

Arquitectura

Cuatro mundos.
Una red.

Red segmentada en subredes departamentales con DMZ aislada para servicios públicos.
